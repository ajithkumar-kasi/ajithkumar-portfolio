# Ajithkumar K — Portfolio

This is a React single-page application for Ajithkumar K's portfolio, built with Vite and styled primarily with Tailwind CSS via CDN plus a few custom CSS utilities.

## Getting Started

From the `portfolio` folder:

```bash
npm install
npm run dev
```

Then open the printed local URL (usually http://localhost:5173) in your browser.

## Production Build

```bash
npm run build
npm run preview
```

- Icons are rendered using the Lucide UMD bundle included in `index.html`.
- Tailwind styles are loaded via the CDN script tag, so no Tailwind build step is required.
